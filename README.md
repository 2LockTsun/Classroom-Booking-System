# Classroom Booking System

This is a project for my SBA.

## Basic Start Up Guide

Open two terminals

#### Front end

```bash
cd src/front-end
npm i
npm run start
```

Then open the website https://localhost:3000 in your browser.

#### Backend

```bash
cd src/back-end
npm i
npm run start
```
