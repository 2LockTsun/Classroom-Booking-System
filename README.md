## Basic Start Up Guide

#### Environment
Make sure you have installed [node.js with npm]() before you start to use this project.

#### Front end

```bash
cd src/front-end
npm install
npm run start
```

Then open the website https://localhost:3000 in your browser.

#### Backend

```bash
cd src/back-end
npm install
npm run start
```
